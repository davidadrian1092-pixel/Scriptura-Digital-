# Scriptura-Digital-
Scriptura Digitală este o platformă web interactivă și educațională dedicată studierii, analizării și explorării textului biblic. Aplicația oferă o experiență de lectură fluidă, fiind completată de unelte avansate de accesibilitate și o hartă istorică interactivă și un modul integrat pentru evaluarea cunoștințelor prin teste tip quiz.
